Install

As of v3, Skin is distributed as a package on the public NPM repository.

The package can be installed to the node_modules folder of your working directory using the npm install command:

{% highlight html %} npm install skin-ebay {% endhighlight %}

Or alternatively by adding the package as a dependency in a package.json file and then running `npm install`:

{% highlight json %} { "dependencies": { "skin-ebay": "~3" } } {% endhighlight %}

As of v3, Skin follows standard semantic versioning.

Lasso

Skin is optimised for use with Lasso.js.

Include the skin-ebay package in your browser.json to include the entire set of Skin modules:

{% highlight json %} { "dependencies": [ "skin-ebay" ] } {% endhighlight %}

Or alternatively by specifying modules on a per-need basis:

{% highlight json %} { "dependencies": [ "skin-ebay/core", "skin-ebay/button", "skin-ebay/iconfont", "skin-ebay/grid", "skin-ebay/less" ] } {% endhighlight %}

Skin provides the following modules:

  • skin-ebay/actionable
  • skin-ebay/button
  • skin-ebay/card
  • skin-ebay/checkbox
  • skin-ebay/combo
  • skin-ebay/core
  • skin-ebay/dialog
  • skin-ebay/field
  • skin-ebay/form
  • skin-ebay/global
  • skin-ebay/grid
  • skin-ebay/iconfont
  • skin-ebay/icon
  • skin-ebay/less*
  • skin-ebay/listbox
  • skin-ebay/marketsans
  • skin-ebay/menu
  • skin-ebay/pagination
  • skin-ebay/radio
  • skin-ebay/spinner
  • skin-ebay/svg
  • skin-ebay/switch
  • skin-ebay/tab
  • skin-ebay/textbox
  • skin-ebay/utility

* The skin-ebay/less module requires the additional lasso-less plugin in your app's package.json dependencies.
